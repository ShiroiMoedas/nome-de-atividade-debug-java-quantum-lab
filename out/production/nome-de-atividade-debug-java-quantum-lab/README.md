# nome-de-atividade-debug-java-quantum-lab
Atividade para debugar e avaliar codigo java com erros, apontar erros.

## Objetivo_Código: O código tem o objetivo de inserção do nome do usuário, exigindo o nome 1 vez. Após exibir o nome, o código acessará um menu de escolhas, onde ele receberá 3 opções;
---
1 - Para soma
2 - Para subtração 
3 - Sair
Onde ele fará as operações.
